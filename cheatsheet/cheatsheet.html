Pi Zero 2W · PICO-8 · Keyboard & API Reference

Keyboard Shortcuts

Navigation

RSHIFT + ← Home
RSHIFT + → End
RSHIFT + ↑ Page Up
RSHIFT + ↓ Page Down
RSHIFT + [ Ctrl+Home (file start)
RSHIFT + ] Ctrl+End (file end)
Ctrl + ← / → Word jump
Alt + F / B Word jump fwd/back (readline)

Selection

LSHIFT + arrow Select character
LSHIFT + RSHIFT + ← / → Word select (Ctrl+Shift+arrow)
LSHIFT + RSHIFT + [ Select to line start
LSHIFT + RSHIFT + ] Select to line end

Terminal

RSHIFT + A Ctrl+A — line start (readline)
RSHIFT + E Ctrl+E — line end (readline)
RSHIFT + U Ctrl+U — delete to line start
RSHIFT + K Ctrl+K — delete to line end
RSHIFT + W Ctrl+W — delete word back
RSHIFT + L Ctrl+L — clear screen
RSHIFT + C Ctrl+C — interrupt
RSHIFT + D Ctrl+D — EOF
RSHIFT + Z Ctrl+Z — suspend
RSHIFT + key Ctrl+key (any key)
LSHIFT + RSHIFT + key Ctrl+Shift+key

Mouse Mode

F5 Toggle mouse mode
D-pad Move cursor
[ Left click
] Right click

PICO-8 API

Callbacks

_init() Run once on cart start
_update() Logic tick, 30Hz
_update60() Logic tick, 60Hz
_draw() Render after each update

Graphics

cls([c]) Clear screen to colour c
color([c]) Set default draw colour
pset(x,y,[c]) Set pixel
pget(x,y) Get pixel colour
line(x0,y0,x1,y1,[c]) Draw line
rect(x0,y0,x1,y1,[c]) Rectangle outline
rectfill(x0,y0,x1,y1,[c]) Filled rectangle
circ(x,y,r,[c]) Circle outline
circfill(x,y,r,[c]) Filled circle
rrect(x0,y0,x1,y1,r,[c]) Rounded rect outline (0.2.7+)
print(str,[x,y,[c]]) Print text; returns end x
camera([x,y]) Set camera offset
clip([x,y,w,h]) Set clipping region
pal([c0,c1,[p]]) Remap colour c0→c1; p=0 display, p=1 sprite
palt([c,t]) Set colour c transparency (true/false)
fillp([pat]) Set fill pattern

Sprites

spr(n,x,y,[w,h],[fx],[fy]) Draw sprite n at (x,y); w,h in tiles; fx,fy flip
sspr(sx,sy,sw,sh,dx,dy,[dw,dh],[fx],[fy]) Stretch blit from sprite sheet

Map

map(mx,my,sx,sy,w,h,[layer]) Draw map region to screen
mget(x,y) Get tile at map cell
mset(x,y,v) Set tile at map cell

Input

btn([i,[p]]) True while button i held; p=player (0/1)
btnp([i,[p]]) True on first press + repeat; 15f delay, 4f interval
0=← 1=→ 2=↑ 3=↓ 4=O 5=X Button indices

Audio

sfx(n,[chan,[offset]]) Play SFX n; n=-1 stop all
music([n,[fade,[mask]]]) Play music pattern n; n=-1 stop

Math

flr(x) Floor (round toward -∞)
ceil(x) Ceiling
rnd(x) Random number 0..x
abs(x) Absolute value
min(x,y) / max(x,y) Min / max of two values
sgn(x) Sign: -1, 0, or 1
sqrt(x) Square root
cos(x) / sin(x) Trig — input 0..1 turns (not radians!)
atan2(dx,dy) Angle as 0..1 turns

Tables

add(t,v,[i]) Append v to table t (or insert at i)
del(t,v) Remove first matching value from t
deli(t,i) Remove element at index i
count(t,[v]) Count elements (or matching v)
all(t) Array iterator for for..in
pairs(t) Key-value iterator

Strings

chr(n) Number to character
ord(str,[i]) Character to number
tostr(v,[fmt]) Convert value to string
tonum(str) Convert string to number
sub(str,i,[j]) Substring from i to j

Memory

peek(addr) / poke(addr,v) Byte read / write
peek2/poke2, peek4/poke4 16-bit / 32-bit fixed-point R/W
@addr / %addr / $addr Short-form: byte / 16-bit / fixed-point read
memcpy(dest,src,len) Copy memory region
stat(n) Query system state (see index in full manual)

System

menuitem(i,[label,fn]) Add item to pause menu (i=1..5)
poke(0x5F2D, 1) Enable devkit input (mouse + raw keyboard)
stat(30) / stat(31) Keypress available / last key char
stat(32) / stat(33) Mouse X / Y position
stat(34) Mouse buttons (bit0=L, bit1=R, bit2=M)
